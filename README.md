# Python Assignment — Part 3: File I/O, APIs & Exception Handling

## Overview
This project fetches real product data from a public API, processes it, saves results to files, and handles all failure scenarios gracefully.

## Tasks Covered
- Task 1: File Read & Write Basics
- Task 2: API Integration using DummyJSON
- Task 3: Exception Handling
- Task 4: Logging to File

## How to Run
Open part3_api_files.ipynb in Google Colab or any Jupyter environment and run all cells.
